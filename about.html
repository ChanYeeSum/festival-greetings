✦ 节日烟花祝福 ✦

为特别的TA，送上一份璀璨的祝福

项目简介

节日烟花祝福是一个开源的在线祝福生成器，让用户可以轻松创建精美的节日祝福卡片并分享给亲朋好友。

支持多种节日主题（新年、春节、中秋、情人节等），配合绚丽的烟花动画效果，让每一份祝福都独一无二。

功能特色

烟花动画

Canvas 绘制的绚丽烟花，支持自定义密度和效果

多节日主题

新年、春节、中秋、情人节等多种节日风格

链接分享

生成专属链接，支持 Base64 编码，更短更美观

二维码分享

生成二维码，方便扫码查看祝福

动画设置

自定义烟花密度、星空亮度等效果

多语言支持

支持中文和英文，随时切换

技术栈

本项目采用纯前端技术实现，无需后端服务，部署简单。

HTML5 CSS3 JavaScript Canvas Base64 GitHub Pages

URL 参数格式

分享链接支持两种格式：Base64 编码格式（推荐）和普通参数格式（兼容旧版）。

Base64 编码格式将所有参数压缩为一个短字符串，链接更短更美观，特别适合在社交平台分享。
